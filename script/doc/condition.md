# 条件变量

条件变量是在变量种设置的，决定某个条件是否生效的参数。

一般用逗号分割，>结束,可以加空格带参数，可以加感叹号表示否操作。

比如
```
maxexp 50000>mq
!maxexp 50000>fish
```

目前主要支持条件变量的变量包括

* combat
* command
* quest

## 最大经验 maxexp

参数为最大经验，用法可以参考#noob别名

## 阅历 yueli

参数为最低经验，用法可以参考#noob别名

## 潜能 pot

参数为最大潜能

## 战斗类型 ctype

参数为战斗类型，一场战斗只有一个类型，具体战斗类型可以参考每次战斗时会刷屏的信息

## 战斗标签 ctag
参数为战斗类型，一场战斗可以有多个标签，比如爬塔每10层会多一个标签
